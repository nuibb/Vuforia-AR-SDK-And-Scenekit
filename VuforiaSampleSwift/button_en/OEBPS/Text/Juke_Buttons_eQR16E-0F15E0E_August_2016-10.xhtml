SEEK/TRACK BUTTON

AUDIO SYSTEM (WITHOUT NAVIGATION)

FM radio mode

Pressing the button starts the tuning mode. The radio tuner seeks from low to high frequencies and stops at the next broadcasting station. During seek mode, the audio output is muted. If no broadcasting station can be found within the complete band cycle, it will return to the initial frequency.

DR (Digital Radio) mode

Pressing the button will select the next station.

CD and AUX mode

When the button is pressed shortly, the next track from a CD or externally connected device will be played.

When the button is pressed continuously, the track will be played at high speed. When the button is released, the track will be played at normal playing speed again.

TELEPHONE AND RADIO BUTTONS

COMBINED AUDIO AND NAVIGATION SYSTEM (“NISSAN CONNECT”)

button:

Access the telephone menu:

  • [Phonebook/Contacts] Select the name from an entry stored in the phone book.
  • [Call Lists] Select the name/number from the incoming, outgoing or missed calls list.
  • [Redial] Redial the last outgoing call from the vehicle.
  • [Keypad] Input the phone number manually using the keypad displayed on the screen.

RADIO button

  • Change radio transmission source

    Press the <RADIO> button to change the radio band as follows: FM1  FM2  DAB (Digital Audio Broadcast)  FM1. When the <RADIO> button is pressed, the radio will come on at the last received radio station. If audio is already playing using one of the other input sources (iPod, Bluetooth, USB, AUX-in) pressing the <RADIO> button will switch off the playing source mode and the last received radio station will be selected.

  • Automatically setting preset stations

    When the <RADIO> button is pressed for more than 1.5 seconds, the six stations with the strongest signals are stored in the preset (1 to 6) buttons of the band. During the search, a notification message [Updating FM List] or [Updating DR List] appears in the display and the sound is muted until the operation is complete. Once completed, the radio selects preset button 1.

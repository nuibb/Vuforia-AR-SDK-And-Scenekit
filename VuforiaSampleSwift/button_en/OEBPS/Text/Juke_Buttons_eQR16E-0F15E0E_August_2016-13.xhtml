STATION PRESET BUTTONS 1-6, REPEAT TRACKS, MIX TRACKS AND INPUT CHARACTERS

AUDIO SYSTEM (WITHOUT NAVIGATION)

Radio mode

1-6: Station preset buttons. Pressing a preset button for less than 2 seconds will select a stored radio station. Pressing a preset button for more than 2 seconds will cause the station currently being received to be stored against that preset button.

Media mode:

RPT: Push the <RPT> button and the current track will be played continuously.

MIX: Push the <MIX> button and all the tracks will be played in random order.

A-Z: When a CD with recorded music information tags (ID3â€“text tags) is being played from list view mode, a quick search can be performed to find a track from the list.

1. Press the <ENTER> button to display the available tracks in a listed view mode.

2. To select a track from the list, turn the <TUNE/MENU> dial then press <ENTER>.

3. To find a track from the list, push the <A-Z> button, then turn the <TUNE/MENU> dial for the first alphabetic/ numerical letter of the track title and press <ENTER>. When found, a list of the available tracks will be displayed. When there is no match, the display shows [No match]. If a match is found, select the desired track and press Radio mode to play the selected track.

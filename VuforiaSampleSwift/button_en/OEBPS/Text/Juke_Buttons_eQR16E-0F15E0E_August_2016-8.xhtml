TA BUTTON

AUDIO SYSTEM (WITHOUT NAVIGATION)

Press the <TA> button to activate/deactivate the Traffic Announcement (TA) interrupt function. [TA] will be shown in the display when the Traffic Announcement function is active. The TA function operates in FM/DR (Radio), CD and AUX mode.

When TA is activated and a traffic announcement is received, playing the current audio source will be interrupted and the announcement is tuned in. The display shows a notification message with the radio station name e.g. [TA: Radio 1].

Once the traffic announcement has finished, the unit returns to the source that was active before the traffic announcement was tuned in.

If <TA> is pressed during a traffic announcement, the traffic announcement is stopped. The TA mode returns to the standby mode and the audio unit returns to the previous source.

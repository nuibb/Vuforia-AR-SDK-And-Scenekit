DISP BUTTON

AUDIO SYSTEM (WITHOUT NAVIGATION)

While a CD with recorded music information tags (CD-text/ID3–text tags) is being played, the title of the played track is displayed. If the title information is not provided then [Track] is displayed.

When the <DISP> button is pressed repeatedly, further information about the track can be displayed along with the track title as follows:

CD:

Track time  Artist name  Album title  Track time

CD with MP3/WMA:

Track time  Artist name  Album title  Folder name  Track time
